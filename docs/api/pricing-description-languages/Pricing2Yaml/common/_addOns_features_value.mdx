- **optional**
- Field type: `bool`, `int` or `str` depending on the `valueType` of the feature

Every addon that you model will have by default all features `defaultValue`. You
can customize it by putting a value in it.

```yaml
features:
  supportPriority:
    defaultValue: LOW
addOns:
  B:
    features:
      supportPriority:
        value: MEDIUM
```
