- **optional**
- Field type: `map`

A map containing the addons of your pricing. Each entry of this map
will be the name of the corresponding addon.

```yaml
addOns:
  awesome_addOn:
  cool_addOn:
    # ...
```

:::info
You have to specify at least `plans` or `addOns`. A combination
of both also works.
:::
