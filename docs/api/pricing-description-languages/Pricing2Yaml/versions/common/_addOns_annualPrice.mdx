- **required**
- Field type: `float`

Price of your addon when billing is set to annual.

```yaml
addOns:
  postmanFlowsBasic:
    annualPrice: 12.00
```

:::info
You have to specify at least `monthlyPrice` or `annualPrice`. A combination
of both also works.
:::
