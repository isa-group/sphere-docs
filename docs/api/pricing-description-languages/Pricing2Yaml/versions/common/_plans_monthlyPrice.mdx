- **required**
- Field type: `float`

Price of your plan when billing is set to monthly.

```yaml
plans:
  PRO:
    monthlyPrice: 9.99
```

:::info
You have to specify at least `monthlyPrice` or `annualPrice`. A combination
of both also works.
:::
