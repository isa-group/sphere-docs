## Advanced search

On the left side of the search page, you have a **Filters** panel where you can further customize your search.


You can sort by:
- Name (The pricing name is case-sensitive. For example, petclinic will appear before Petclinic.)
- Configuration space size (number of subscriptions)
- Number of features
- Number of usage limits
- Number of plans
- Number of add-ons
- Minimun subscription price
- Maximun subscription price

The sort direction determines the order in which items are arranged:
- Ascending: items are arranged from smallest to largest
- Descending: Items are arranged from largest to smallest

Lastly you can focus the search interval of the pricings by:
- Configuration space size (number of subscriptions)
- Minimun subscription price and Maximun subscription price

After customizing your search, you need to press the **Filter** button to perform the search.
